# Installation

## Local development host

1. Open the project in VS Code.
2. Build the extension output (`dist/extension.js`).
3. Press `F5` to start an Extension Development Host.
4. Open `examples/sample.mncnk` and verify highlighting.

## Local VSIX install

1. Package a `.vsix`.
2. In VS Code, open Extensions panel.
3. Use `Install from VSIX...` and select your package.
4. Reload the window and verify `.mncnk` association.
